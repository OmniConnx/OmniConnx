# OmniConnx

## MVP:
### **Features**:
- #### **User profiles**
   - Name 
   - Username
   - Professional title 
   - Location
   - Picture
   - Skills list
   - Personal info for profile

- ### **Posts**

   - Authored by users
   - Users can respond/react 
   - Sorted feed

- ### **Skills** 

   - Skills represented by hashtag (#anatomy, #optometry, #dermatology) 
   - Each individual skill will have its own page 
   - Skills can be linked in posts and on user profiles 

## **Pages**: 

   - Landing page 
   - Log in 
   - User profile 
   - Posts 
   - Skills page 


## How to run server 
In terminal:
'''
nodemon 
'''
